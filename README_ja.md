# dot-maker
数値データ（CSV）、カラーリスト（TXT）、設定ファイル（TXT）を組み合わせて、PNG画像を生成するツールです。

## 必要なファイル構成
一つのフォルダの中に、以下の3つのファイルを用意してください。

### 1. config.txt
1マスのサイズを何ピクセルにするかを記述します。
- 例: 10 （1つの数字が10x10pxの正方形になります）
### 2. colors.txt
カラーコード（#RRGGBB形式）を改行区切りで記述します。行数がデータの数字に対応します。
- 1行目: 数字「1」の色
- 2行目: 数字「2」の色
- …
- **注意:** 「0」は常に透明として扱われるため、記述不要です。

### 3. data.csv
画像の設計図となる数字をカンマ区切りで入力します。
- 例:
  0,1,0
  1,2,1
  0,1,0

## 動作環境
Python 3.x および Pillow (PIL) ライブラリ
インストール: `pip install Pillow`

## 使い方
スクリプトを実行する際、引数として対象のフォルダ名を渡します。

python convert.py [フォルダ名]

実行後、フォルダ内に output.png が書き出されます。
